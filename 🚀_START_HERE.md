## 🔧 Configuration

### Environment Variables
Create a `.env` file in the root:
```env
# Supabase (Required)
SUPABASE_URL=https://your-project.supabase.co
SUPABASE_ANON_KEY=your-anon-key-here
SUPABASE_SERVICE_ROLE_KEY=your-service-role-key-here

# OAuth (Optional - for Google/Microsoft login)
GOOGLE_CLIENT_ID=your-google-client-id
GOOGLE_CLIENT_SECRET=your-google-client-secret
MICROSOFT_CLIENT_ID=your-microsoft-client-id
MICROSOFT_CLIENT_SECRET=your-microsoft-client-secret

# Payment Processing (Optional)
STRIPE_SECRET_KEY=sk_test_...
STRIPE_WEBHOOK_SECRET=whsec_...

# SMS Notifications (Optional)
TWILIO_ACCOUNT_SID=AC...
TWILIO_AUTH_TOKEN=...
TWILIO_PHONE_NUMBER=+1...
```

## 🐛 Troubleshooting

### Issue: npm install fails
**Solution:** Try `npm install --legacy-peer-deps`

### Issue: Port 5173 already in use
**Solution:** Run `npm run dev -- --port 3000`

### Issue: Charts not rendering
**Solution:** Ensure browser window is wide enough (Recharts needs space)

### Issue: Fonts not loading
**Solution:** Check internet connection (Acme font loads from Google Fonts)

### Issue: Navigation not working
**Solution:** Clear browser cache and restart dev server
